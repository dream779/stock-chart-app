# 美股基金行情看板首屏优化设计

## 背景

当前首页同时展示两张指数卡片和两张历史走势图，上半部分（标题 + 卡片 + 时间切换栏）占用空间过大，导致用户首次进入页面时无法看到下方的图表，需要滚动。

## 目标

1. 首屏无需滚动即可看到完整的历史走势图。
2. 精简上半部分信息，移除无用内容。
3. 只展示一个图表，默认选中标普 500（^GSPC）。
4. 点击顶部卡片可切换到纳斯达克 100（^NDX）。

## 方案

采用「极简卡片 + 图表占满剩余视口」方案。

## 详细设计

### 页面结构

- 移除页面顶部 `<h1>` 主标题「美股基金行情看板」和 `<p>` 副标题。
- 顶部保留两张指数卡片并排展示；在移动端自动堆叠。
- 默认「标普 500」为激活态。

### 指数卡片（IndexCards）改造

- 新增 props：
  - `selectedSymbol: string`：当前选中的指数代码。
  - `onSelect: (symbol: string) => void`：点击卡片时的回调。
- 卡片可点击，点击非激活卡片切换选中状态。
- 激活卡片使用蓝色边框 / 浅蓝背景做视觉区分。
- 内容精简：
  - 第一行：指数名称 + 「· 更新于 YYYY-MM-DD」（小字灰色）。
  - 右上角保留迷你市场状态标签（如 CLOSED），字号更小、背景更浅。
  - 第二行：大号当前价格 + 涨跌幅标签。
  - 移除股票代码（^GSPC / ^NDX）。
- 整体减小 padding、字号、行高，降低单张卡片高度。

### 图表区域（page.tsx）改造

- 使用 flex 布局，让图表容器占满 `100vh - 卡片高度 - 时间切换栏高度` 的剩余空间。
- 只渲染当前选中指数的单个 `Chart` 组件。
- 图表标题显示为「标普 500 - 最近1年」等，不显示代码。
- 保留时间范围切换按钮（最近1周 / 1个月 / 3个月 / 最近1年）。
- 保持现有的历史数据请求和统计计算逻辑。

### 交互

- 点击卡片切换选中指数，图表随选中指数和时间范围自动刷新。
- 初始选中 ^GSPC。

## 受影响的文件

- `app/page.tsx`
- `components/IndexCards.tsx`
- `components/Chart.tsx`（可能需要调整高度自适应）

## 验收标准

- [ ] 首屏可见完整图表区域，无需滚动。
- [ ] 顶部标题、副标题、股票代码已移除。
- [ ] 卡片显示名称、更新时间、价格、涨跌幅、迷你状态标签。
- [ ] 默认展示标普 500 图表。
- [ ] 点击纳斯达克 100 卡片切换到对应图表。
- [ ] 时间范围切换对当前选中指数生效。
